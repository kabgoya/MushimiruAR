# 🐞 ムシミルAR

スマホのカメラで手のひらを写すと、魔法陣から昆虫の3Dモデルが出てくるWebアプリです。
インストール不要で、ブラウザ(Safari / Chrome)だけで動きます。

## あそびかた

1. アプリを開いて「はじめる!」を押す(カメラの使用を「許可」してください)
2. 手のひらをパーに開いてカメラに写す
3. 魔法陣が現れて、中から昆虫が出てきます
4. 画面下の 📸 ボタンで、昆虫を手のひらに乗せた写真が撮れます
5. 🤫 ここだけのヒミツ : 手をすばやく振ると…?(お子さんへのサプライズ機能です)

## 動かしかた

カメラを使うアプリなので、**ファイルをダブルクリックしただけでは動きません**。
以下のどちらかの方法で開いてください。

### 方法A : パソコンで試す(かんたん)

1. [VS Code](https://code.visualstudio.com/) に拡張機能「**Live Server**」を入れる
2. このフォルダを VS Code で開き、`index.html` を右クリック →「Open with Live Server」
3. ブラウザで開いたら「はじめる!」を押す

(または、コマンドが使える人は このフォルダで `npx serve` を実行 → `http://localhost:3000` を開く)

### 方法B : スマホで使う(公開する)

スマホのカメラは **https のページでしか動きません**。無料の公開サービスを使うのが一番かんたんです。

- **Netlify Drop** : https://app.netlify.com/drop にこのフォルダをドラッグ&ドロップするだけ
- **GitHub Pages** : GitHubにアップロードして Settings → Pages で公開「

公開されたURL(https://〜)をスマホで開けば使えます。

> 📶 初回はインターネットから3D表示ライブラリと手認識AIをダウンロードするため、ネット接続が必要です。

## 🔄 昆虫の3Dモデルを差し替える方法(プログラミング不要!)

1. 昆虫の3Dモデルファイルを用意する
   - 対応形式 : **.glb(おすすめ)** / .gltf / .obj
   - 無料モデルは [Sketchfab](https://sketchfab.com/)(ライセンス表記に注意)などで入手できます
2. そのファイルを **`models/` フォルダ**に入れる
3. **`models/config.json`** をメモ帳などで開いて、ファイル名を書き換える

```json
{
  "file": "kabutomushi.glb",   ← 入れたファイルの名前に書き換える
  "sizeCm": 5,                 ← 昆虫の全長をcmで指定(カブトムシなら5)
  "scale": 1.0,                ← 大きさの微調整(2.0 で2倍、0.5 で半分)
  "rotationY": 0,              ← 向きの調整(度数。180 で反対向き)
  "offsetYCm": 0               ← 高さの調整(cm。めり込むときに増やす)
}
```

4. ページを読み込み直せば、新しい昆虫が出てきます

### 🐜 実寸表示について

このアプリは、カメラに写った**手のひらの大きさを定規の代わり**にして、昆虫を実物と同じ大きさ(実寸)で表示します。

- モデルが**メートル単位の実寸**で作られている場合(glbの標準。実寸5cmなら0.05の大きさ) → `sizeCm` は `null` のままでOK。そのまま実寸で表示されます
- モデルの単位や大きさがよくわからない場合 → `sizeCm` に実物の全長(cm)を書けば、自動でその大きさにそろえてくれます

※ モデルファイルが無い・読み込めない場合は、自動で実寸1cmのテントウムシが表示されます。
※ .obj は色情報が入っていないことが多く灰色で表示されます。.glb をおすすめします。

## うまく動かないとき

| 症状 | 対処 |
|---|---|
| カメラが映らない | https(または localhost)で開いているか確認。ブラウザのカメラ許可を確認 |
| 「じゅんびちゅう…」のまま | インターネット接続を確認して、ページを読み込み直す |
| 魔法陣が出ない | 手のひらを**パーに開いて**、明るい場所でカメラに写す |
| 昆虫が大きすぎ/小さすぎ | `models/config.json` の `scale` の数字を調整する |
| 動きがカクカクする | 他のアプリやタブを閉じる。新しめのスマホ推奨 |

## ファイル構成

```
MushimiruAR/
├── index.html      … ページ本体(説明ウィンドウ・トグルスイッチ)
├── css/style.css   … 見た目の設定
├── js/
│   ├── main.js         … アプリ全体の司令塔(状態管理・振り判定)
│   ├── camera.js       … カメラの起動・切り替え
│   ├── hand-tracker.js … 手のひらの検出(MediaPipe)
│   └── ar-scene.js     … 3D表示(魔法陣・昆虫・アニメーション)
├── models/
│   ├── config.json … ★モデル差し替え用の設定ファイル
│   └── (insect.glb) … ★ここに昆虫の3Dモデルを入れる
└── README.md       … この説明書
```

## 使用ライブラリ(CDNから自動読み込み)

- [Three.js](https://threejs.org/) — 3D表示
- [MediaPipe Hand Landmarker](https://developers.google.com/mediapipe) — 手のひら検出
